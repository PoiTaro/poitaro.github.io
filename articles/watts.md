## ワッツオンラインストアの「送料」はいくら？

オンラインストアで一番気になるのが送料ですよね。ワッツの送料体系は「自宅などへの配送」と「店舗受取」の2パターンで大きく異なります。

### 自宅などへ配送する場合の送料

まずは、通常の「配送」の場合です。

* **全国一律：770円（税込）**
* **北海道：1,540円（税込）**
* **沖縄県：2,780円（税込）**

100円の商品を買うのに送料770円は、正直ちょっとためらってしまいますよね…。でも、ご安心ください！送料無料にする方法もしっかり用意されています。

#### 配送で送料無料にする条件

配送の場合、**1回の注文合計金額が7,700円（税込）以上**になると、送料が無料になります。

ただし、注意点がひとつ！沖縄県だけは、7,700円以上購入した場合でも2,200円（税込）の送料がかかってしまうんです。北海道にお住まいの方は、7,700円以上で無料になるのでご安心ください。

また、クーポンを利用する場合は、**クーポン適用後の金額が7,700円（税込）以上**である必要があるので、計算する際は気をつけてくださいね。

「7,700円も買うものがないよ…」という方にこそ、次にご紹介する「店舗受取」がおすすめです！

## 【必見】送料無料で超便利！「店舗受取」のメリットと使い方

「送料をどうしても払いたくない！」という方のための最強のサービスが、この「店舗受取」です。

なんと、**1回の注文合計金額が990円（税込）以上**であれば、**全国の対応店舗で送料無料**で商品を受け取れるんです！

990円（税込）未満の注文はできませんが、100円ショップでのお買い物なら、10点ほど選べばあっという間にクリアできる金額ですよね。これは使わない手はありません！

### 店舗受取のメリット

このサービスのメリットは、送料無料だけじゃありません。

* **メリット1：なんといっても送料無料！**
    990円（税込）以上の注文という条件だけで、あの配送料がゼロになります。
* **メリット2：近くの店舗にない商品が買える**
    「オンラインストアで見て気になってたけど、地元のワッツには置いてなかった…」そんな商品も、オンラインで注文して店舗で受け取れます。
* **メリット3：好きなタイミングで受け取れる**
    宅配便のように「いつ届くかな…」と家で待機する必要がありません。店舗の営業時間内なら、あなたの都合のいい時に受け取りに行けます。
* **メリット4：対応店舗が全国に拡大中！**
    2024年2月からサービスが拡大し、全国500店舗以上で利用可能になっています。あなたのお近くのワッツも対象かもしれません。

### 店舗受取の使い方（利用手順）

使い方はとっても簡単です。

1.  ワッツオンラインストアで欲しい商品をカートに入れます。
2.  注文画面に進み、配送方法で「店舗で受け取る」を選びます。
3.  商品を受け取りたい店舗を検索して選択します。
4.  注文を確定します。
5.  注文確定後、商品が発送されると「発送通知メール」が届きます。
6.  発送通知メールが届いてから3日後以内（または3〜5日後）に、指定した店舗に商品が到着します。
7.  店舗のレジなどで、店員さんに「オンラインショップの商品を受け取りに来た」旨を伝え、「名前」と「注文番号」を提示して商品を受け取ります。

### 店舗受取の注意点（デメリット）

とても便利な店舗受取ですが、いくつか注意点もあります。

* **注文後の受け取り店舗変更は不可**
    一度注文を確定すると、受け取り店舗や配送先を変更することはできません。
* **受け取りには期限がある**
    店舗での保管期間は、**店舗に商品が到着した日を含めて30日間**です。この期間を過ぎると自動的にキャンセル扱いになり、場合によっては手数料などを請求される可能性もあるので、必ず期限内に受け取りに行きましょう。
* **税込990円未満の注文はできない**
    このサービスを利用するための最低注文金額です。

## ワッツオンラインストアの「支払い方法」まとめ

最後に、ワッツオンラインストアで利用できる支払い方法をご紹介します。実店舗とは使える決済方法が異なるので注意してくださいね。

### 利用可能な支払い方法一覧

| 支払い方法の種類 | 具体的なサービス名 |
| :--- | :--- |
| **クレジットカード** | VISA / MASTER / JCB / American Express (※一括払いのみ) |
| **スマホ決済（QRコード）**| PayPay / 楽天ペイ / d払い |
| **キャリア決済** | auかんたん決済 / ソフトバンクまとめて支払い |
| **その他決済** | Google Pay / Apple Pay / エポスかんたん決済 |
| **後払い決済** | Paidy (ペイディ) |
| **法人・個人事業主向け** | NP掛け払い（銀行振込） |

### おすすめの支払い方法は？

これだけ種類があると迷ってしまいますが、僕のおすすめはやはり**クレジットカード**か**PayPayなどのスマホ決済**です。

理由は、決済手数料が無料であることと、それぞれの決済サービス側でポイントが貯まるからです。ワッツオンラインストアでお得に買い物しつつ、ポイ活もできちゃうわけですね。

### 支払い方法の注意点

ワッツオンラインストアの支払い方法に関する注意点は以下の通りです。

* **代金引換（代引き）は使えない**
    「商品と引き換えに現金を払いたい」という方は多いかもしれませんが、残念ながらオンラインストアでは対応していません。
* **銀行振込は「NP掛け払い」のみ**
    個人の方が利用できる一般的な銀行振込（前払い）はありません。「NP掛け払い」は法人・個人事業主向けのサービスとなります。
* **クレジットカードは一括払いのみ**
    分割払いやリボ払いには対応していないのでご注意ください。

支払い方法も充実しているので、あなたの使いやすい決済方法がきっと見つかるはずです。
